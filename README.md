# ABACUS 官方网站

现代化、震撼视觉效果的 ABACUS 第一性原理计算软件包官方网站

## 🎨 全新设计亮点

### 震撼的Hero区域
- **分栏式布局**：左侧内容 + 右侧Logo可视化
- **动态Logo展示**：官方Logo带浮动动画和光晕效果
- **渐变标题**：超大"ABACUS"标题配合"原子算筹"中文名
- **特性徽章**：高性能并行、多基组支持、AI集成三大核心特点
- **背景动效**：动态渐变光球营造科技感
- **双CTA按钮**：立即下载 + Star on GitHub

## 🚀 技术栈

- **Vue 3**：现代化前端框架
- **Vite**：快速构建工具
- **Font Awesome 6**：专业图标库
- **Inter 字体**：现代化字体
- **纯 CSS**：无需额外框架

## 📦 安装和运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 🎨 设计亮点

### 配色方案
- 主色：`#2c5aa0` (蓝色)
- 次色：`#5b9bd5` (浅蓝)
- 高亮色：`#ff9500` (橙色) - FAQ专用

### 视觉效果
- **渐变背景**：Hero区域使用渐变叠加
- **毛玻璃效果**：统计卡片、按钮使用backdrop-filter
- **阴影层次**：多层次阴影营造深度
- **圆角设计**：所有卡片使用圆角

### 动画效果
